HEY!

Welcome to the interactive wishlist. If you are in search for a present for me you are right here.

How does this work?

In the lower half of this website a google docs document is going to load. Feel free to use it if you only want to have a look around, or just search for ideas. If at some point you decided on something from the list, mark it as bought.
To do that, use this link to access the original document. Please don't write or delete anything. If you want to mark something as bought, mark it as crossed out. To do that mark the text and press "alt+shift+5" ("⌘ + Shift + X" on MAC) or use the menu via "Format > Text > Strikethrough".
When I actually received that present I will completely take it out of the list. The local copy on this website will update automatically within a couple of minutes/ after refreshing the website.
