HEY!

Welcome to the interactive wishlist. If you are in search for a present for me you are right here.

How does this work?

In the lower half of this website a google docs document is going to load. Feel free to use it if you only want to have a look around, or just search for ideas. If at some point you decided on something from the list, mark it as bought.
To do that, use this link to access the original document. You will have commenting rights in that document. Just delete what you bought. This will then appear as deleted text so that everyone knows it was bought already. When I actually received that present I will completely take it out of the list. The local copy on this website will update automatically within a couple of minutes.
